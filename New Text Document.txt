# 🏫 SaaS School Management System (ERP + Website Builder)

## 📌 Project Overview

This project is a **Multi-Tenant SaaS School Management System** built using:

* **Backend:** Laravel (API-based)
* **Frontend:** React.js
* **Database:** MySQL

The system allows a central admin (Super Admin) to create and manage multiple schools.
Each school gets:

* A dedicated ERP system
* A customizable website
* Subscription-based access
* Module-based feature control

---

## 🎯 Objective

To provide a **single scalable platform** where schools can:

* Manage daily operations (students, staff, fees, exams)
* Automatically generate a website
* Access features based on subscription plans

---

## 🏗️ System Architecture

### 🔹 Multi-Tenant Structure

* Single database (initial phase)
* All tables contain `school_id`
* Data is isolated per school

## 💳 Subscription Logic

* Plan defines:

  * Feature access
  * Limits (students, modules)
* Expiry disables access
